# Lab Environment Setup

## Virtualization

- VirtualBox used for lab environment
- Ubuntu Server 22.04 LTS deployed as SOAR host
- 4 GB RAM, 2 CPU cores, 40 GB disk

## SOAR Deployment

- Docker and Docker Compose installed
- Shuffle SOAR platform deployed using official GitHub repository
- Containers verified running via `docker ps`

## Access

SOAR platform accessed via browser on port 3001 using the VM's internal IP address.

## Security Measures

- System updated and hardened before deployment
- Admin password changed after first login
